# 🛒 Amazon Clone (HTML & CSS)

This project is a **frontend clone of the Amazon homepage** built using **HTML and CSS only**.
It was created to practice **layout design, styling, and responsive UI development**.

🌐 **Live Demo**
https://amazonclone-dusky-seven.vercel.app/

---

## 🚀 Features

* Amazon-style navigation bar
* Product sections layout
* Clean UI similar to Amazon
* Responsive design using CSS
* Beginner-friendly frontend project

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
